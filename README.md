# 🎮 Perfect Parry

**리듬 기반 패링 액션 게임**

## 🎯 게임 설명

적의 공격을 정확한 타이밍에 터치하여 패링하는 리듬 게임입니다!

### 핵심 메커니즘
- **PERFECT** (±50ms): 3x 데미지 + 300 점
- **GOOD** (±150ms): 2x 데미지 + 200 점
- **OK** (±300ms): 1x 데미지 + 100 점
- **MISS**: 체력 감소 + 콤보 리셋

### 컨트롤
- **모바일**: 화면 터치
- **PC**: 스페이스바

## 🚀 실행 방법

### 로컬에서 실행
```bash
# 1. 폴더로 이동
cd perfect-parry

# 2. 로컬 서버 시작 (선택사항, 브라우저에서 직접 열어도 됨)
python3 -m http.server 8000

# 3. 브라우저에서 열기
# http://localhost:8000
```

### 브라우저에서 바로 실행
`index.html` 파일을 브라우저로 열기만 하면 됩니다!

## 📱 모바일 실행

1. 파일을 모바일로 전송
2. 모바일 브라우저에서 `index.html` 열기
3. 터치로 플레이!

## 🎨 커스터마이징

### BPM 변경
`game.js` 에서 `this.bpm` 값 수정:
```javascript
this.bpm = 120;  // 더 빠르게: 140, 더 느리게: 100
```

### 타이밍 윈도우 조정
```javascript
this.PERFECT_WINDOW = 0.05;  // ±50ms
this.GOOD_WINDOW = 0.15;     // ±150ms
this.OK_WINDOW = 0.30;       // ±300ms
```

### 보스 패턴 변경
```javascript
this.bossPattern = [0, 2, 4, 6, 8, 11, 13, 16, 18, 20, 23, 25, 28, 30, 32, 35, 37, 40];
// 비트 번호 배열 - 숫자를 추가/삭제하여 패턴 변경
```

## 🐛 문제 해결

### 게임이 실행되지 않음
- 브라우저 콘솔 (F12) 에서 에러 확인
- 인터넷 연결 확인 (Phaser CDN 로드 필요)

### 터치 반응 없음
- 모바일 브라우저에서 실행
- `user-scalable=no` 메타 태그 확인

### FPS 낮음
- 간단한 패턴으로 테스트
- 브라우저 하드웨어 가속 확인
